## 포인터 변수의 선언
- 형식 : 자료형 * 포인터변수명;
- `int *p`
- 변수 p는 포인터 변수로서 정수형의 자료를 갖는 변수의 주소를 갖는다
- p: 포인터 변수로서 정수형 자료가 수록되어 있는 주소
- \*p: 해당 주소에 수록되어있는 정수형 자료 자체

<img width="487" alt="스크린샷 2023-05-14 오후 7 59 40" src="https://github.com/jinia91/TIL/assets/85499582/4f9f6da0-5c2a-46a9-a21a-0c559197b15a">

> 정리!
> 일반변수 a에서 주소값 추출 -> &a
> 주소값을 변수에 할당할때는 포인터 변수사용 -> \*p = a
> 포인터 변수에서 주소를 볼때는 그냥 변수호출 p
> 포인터 변수에서 주소의 값을 볼때는 \*p
> 포인터 변수의 값을 변수에 할당하려면 b = \*p 겠지?

## 이중 포인터
- 형식 : int **p;
- 포인터 변수에 다시 포인터를 지정하는것으로 포인터에 대한 포인터를 의미

p: 기억공간주소
\*p: 주소 p에 수록되어있는 값, 이 값이 주소로 사용
\*\*p: 주소가 \*p 인 곳에 수록되어있는 값, 즉 int형 자료
